# 🧠 Neural Network Projects Portfolio

## 📌 Overview

This repository is a curated collection of **Artificial Neural Network (ANN) projects** focused on solving diverse real-world problems using tabular and structured data.

Unlike typical ML repositories that focus only on prediction accuracy, this repo emphasizes:

✔ Understanding **how models work**
✔ Applying ANN to **different problem types**
✔ Building intuition around **data, errors, and trade-offs**

---

## 🎯 Purpose of This Repository

The goal of this repository is to demonstrate a **holistic understanding of neural networks**, beyond just training models.

### 🔍 Key Objectives:

* Explore ANN across multiple domains:

  * Regression
  * Classification
  * Anomaly Detection
  * Data Imputation

* Understand **when neural networks work well—and when they don’t**

* Build strong intuition around:

  * Model behavior
  * Error analysis
  * Threshold tuning
  * Data distribution

* Showcase **practical ML thinking** instead of just model performance

---

## 📂 Projects Included

---

### 🍷 1. Wine Quality Prediction

* Regression using ANN
* Compared with Linear Regression
* Insight: simpler models can outperform ANN on tabular data

---

### 🔥 2. Outlier Detection using Autoencoder

* Unsupervised anomaly detection
* Reconstruction error-based approach
* Explored threshold vs detection trade-off

---

### 🧩 3. Missing Value Imputation

* Autoencoder-based imputation
* Artificial masking for correct evaluation
* Significant improvement over baseline methods

---

### 🧠 4. Decision Boundary Visualization

* Visualized ANN decision regions
* Demonstrated nonlinear separability
* Improved interpretability of neural networks

## 🧠 Key Learnings

* Neural networks are **not always the best choice** for tabular data
* Proper **evaluation and baselines are essential**
* Threshold selection is critical in anomaly detection
* Representation learning enables solving **non-traditional ML problems**
* Visualization improves **model interpretability**

## 🚀 How to Use This Repository

Each project is organized in its own folder with:

* Source code
* Individual README
* Clear instructions to run

Clone the repository:

```bash
git clone <your-repo-link>
cd neural-network-projects
```

Run any project:

```bash
cd project-folder
python file.py
```
